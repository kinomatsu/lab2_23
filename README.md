# Постановка задачи (Вариант 23)

## Задача 1. Калькулятор среднего
Ввод N оценок. Вычислить среднее арифметическое, средний балл (по 5-балльной шкале).

## Задача 2. Анализ оценок
Подсчитать количество: пятёрок, четвёрок, троек, двоек. Определить процент успеваемости.

## Задача 3. Диаграмма оценок
В PictureBox нарисовать круговую диаграмму: секторы для каждой оценки, размер пропорционален количеству.

# Особенности реализации
- Валидация вводимых данных (проверка на числа от 2 до 5)
- Использование ListBox для вывода статистики
- Визуализация данных с помощью Graphics в PictureBox
- Разные цвета секторов для разных оценок
